Quarter 4 Projects

Quiz 1: BMI Calculator

Objective: To create a BMI calculator using JavaScript that computes the Body Mass Index based on the user’s height and weight.
Goal: Learn how to use variables, arithmetic operations, conditional statements, and outputs in a real-world application.
Features: User input validation, BMI formula implementation, and category detection (underweight, normal, overweight, obese).

Quiz 2: Weight App

Objective: Build a JavaScript-based program that processes the user’s weight and evaluates results using conditions.
Goal: Understand and apply JavaScript logic for decision-making using if-else statements.
Features: Form handling, result output, use of document.write and innerHTML, plus meaningful error handling.
